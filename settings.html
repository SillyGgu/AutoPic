수동 이미지 생성

버튼을 누르면 선택한 Connection Profile이 해당 메시지용 AutoPic 구조화 블록을 만들고, 자동 생성과 같은 이미지 생성 경로로 전송합니다.

🎛️ NovelAI 고급 파라미터

ST의 이미지 생성 소스가 NovelAI일 때만 동작합니다. Character Prompt, UC, cfg_rescale 등 ST 기본 확장이 미지원하는 NAI v4 전용 기능을 제공합니다.
⚠️ 아래 기능들은 모두 AutoPic 서버 플러그인 설치가 필요합니다. 플러그인 없이 사용하면 이미지 생성이 실패할 수 있습니다.
